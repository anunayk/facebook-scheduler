Facebook Scheduler
===================

Required Libraries
------------------
[facepy](https://github.com/jgorset/facepy/), [time](http://docs.python.org/2/library/time.html) 

Usage
------
* Replace POST_TIME and POST_MESSAGE.
* Run script before the scheduled time.
* Forget!
